rui_steer
====================

Usage
-----
```
roslaunch rui_steer rui_steer.launch 

```

Open your browser, and access to:

`http://<your host name>:8000/rui_steer/`

for example : `http://localhost:8000/rui_steer/`

- Control using Joystick
- Add camera topic
- Press Load video button
